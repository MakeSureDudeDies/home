Celery Docs

Basics

"getgenv()" - Get the executor's global environment.

"getreg()" (other aliases: "debug.getregistry()", "getregistry()" )."

I/O

"writefile("name.extension")" - Write a file in the workspace folder with the desired name in a string.

"readfile("name.extension")" - Read a file from the workspace folder with the name in a string.

"loadfile("name.extension")" (other alias: "dofile()" ) - Load a file from the workspace folder with the name in a string (acts as loadstring with the file content in a string.

"delfile(name.extension")" - Delete a file in the workspace folder with the file name in a string.

Registry

"getinstances()" - Get all instances in the registry.

"getnilinstances()" - Get all instances parented to nil in the registry.
